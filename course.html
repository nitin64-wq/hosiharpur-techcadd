Curriculum

In this module, learners are introduced to the basics of computer knowledge, i.e., computer basics including what it is, how it functions, and where it can be found in real life. The learners are presented with various kinds of computer knowledge, computer terms, and hardware as well as software functions.

Under this module, a student will be able to identify the physical parts of a computer that include items like the CPU, monitor, keyboard, mouse, printer, storage media, etc. This will be based on learning how each of these parts functions.

The module under consideration belongs to the category where the basic knowledge about operating systems is provided to students through practical training on how to run and shut down a computer, manage files and folders, and basic troubleshooting to ensure safe handling and operation of a computer.

The students will be able to create, edit, format, and save documents using MS Word software. The topics in this module will be devoted to practicing typing skills, formatting documents, adding pictures and tables in documents, and preparing professional documents in word format.

This module discusses the concept of spreadsheets and data management with MS Excel as a tool. It gives you an opportunity to work with spreadsheets, entering data, formulating basic formulas and functions, formatting cells, and charting calculations.

Under this module, students are able to design a good presentation using MS power point. Some of the topics include how to create a presentation, how to include text, images, animation, or transitions, among many other topics.

The primary objective of this module is to enable learners to use the internet confidently and responsibly. Through this module, learners are provided with an understanding of how to explore the internet via website browsing, search for useful information, and even how to download internet resources, including an understanding of internet services, including email creation for everyday use to make learners feel comfortable when using the internet for personal as well as professional purposes.

Learners will be taught various computer safety practices that keep both data and devices safe. These will include making strong passwords, knowing computer viruses, understanding antivirus tools, making backups, and staying safe from viruses. Learners will also be taught computer etiquette, which will teach them how to use computers responsibly.
